2.2.1. Order Discount

Order discounts are a great way to encourage customers to make a purchase by offering them a special discount on their order. To set up an order discount, follow these steps:

1. Set Discount Name : Provide a clear and descriptive name for the discount. This name should help you and your customers easily identify the purpose of the discount.

2. Choose the Discount Type:We offer two types of discounts:

  • Fixed Amount Discount: For example, you could offer a discount of $10 off a customer’s order. In this case, the discount amount is fixed, regardless of the total order value.
  • Percentage Discount: For example, if you set a 10% discount, and the customer’s total order is $50, they will receive a $5 discount. The discount is calculated as a percentage of the total order amount.

3. Set Discount Amount: If you select a fixed amount, you’ll need to specify the discount value (e.g., $10). If you choose a percentage, you’ll need to specify the discount percentage (e.g., 10%). Remember that these two options are directly related to the order total or percentage chosen.

4. Minimum Order Value: You’ll also need to set a minimum order value. This means customers must reach a certain order amount before they can use the discount. For example, you can set a minimum order value of $50, which means customers can only use the discount if their total purchase is greater than or equal to $50.

Example: Let’s say you offer a 15% discount on orders over $50. A customer purchases a product worth $100 and applies the discount code. They will receive a $15 discount (15% of $100), making their total $85 (before taxes and shipping).

Here are concise tips for setting up effective order discounts:

Quick Tips for Order Discounts:

  1. Use catchy codes: Create memorable and simple discount codes like "SAVE10" or "WELCOME15" for easy sharing.
  2. Combine discounts strategically: Pair order discounts with free shipping to boost customer interest.
  3. Test minimum order values: Experiment with different thresholds to maximize sales while maintaining profitability.
  4. Highlight savings: Clearly show customers how much they’ll save to encourage purchases.
