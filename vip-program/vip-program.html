VIP Program

The VIP Program is an excellent way to enhance customer loyalty and grow your business by rewarding your most valued customers. Follow these simple steps to set up and manage your VIP Program:

Getting Started

  1. Go to Reward-Race Loyalty > VIP Program.
  2. Review the guidelines and tools provided for setting up your VIP process.

4.1. VIP Tier Steps

Create VIP Tier

To create your VIP tiers, click on the "Create VIP Tiers" button.

Once you click this button, the VIP setup flow will guide you through the process:

Tier Entry Conditions
  1. Determine Entry Criteria: Choose how customers will qualify for VIP tiers. Options include:
    • Points Earned: Base entry on the total points customers accumulate.
    • Amount Spent: Base entry on the total amount customers spend.
  2. Set Tier Count: Specify how many tiers you want to create. You can create up to 5 tiers. Setting the number of tiers is required to proceed to the next step.

After configuring these options, click the "Next" button to continue.

Tier Configuration

In this step, you will configure the details of each tier:

  1. Assign Tier Names: Use unique and descriptive names for each tier to easily differentiate them.
  2. Set Minimum Points: Define the minimum requirement for customers to qualify for each tier.

    Note: Based on the entry condition you selected:

    • If you chose Points Earned, specify the required points amount here.
    • If you chose Amount Spent, set the required spending amount here.

  3. Reward Percentage: Set the percentage reward that customers will receive when they reach a specific tier or move to the next tier.
  4. Select Tier Rewards: Choose one or more rewards for each tier. You can select or deselect rewards as needed.

    Note: Before selecting rewards, ensure they are activated under Reward Program > Ways to Earn Points. If rewards are deactivated, they will not appear as options.

  5. Choose an Icon: Select a default icon or upload a custom icon to visually represent each tier.

Additional Information

  1. Review all settings in the summary section to ensure accuracy.
  2. Once satisfied, click the "Create" button to finalize your VIP Program.
